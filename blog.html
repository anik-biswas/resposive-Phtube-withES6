Question 1: Discuss the scope of var, let, and const

Var :

vars are globally scoped. Its value can be changed.

Let :

let is a block scope. It can be updated but not re-declared

const :

const is a block scope. It can not be updated or re-declared

The use cases of null and undefined

Null

Null is basically an assign value given to a variable. The variable which has been assigned as null contains no value

Undefined

Undefined means not defined. So we declare a variable but do not assign a value to it, the variable becomes undefined.
